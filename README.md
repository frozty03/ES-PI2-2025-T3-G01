# Projeto NotaDez

Aplicação web direcionada aos docentes para gerenciamento de instituicoes, turmas, disciplinas, alunos e notas.

## Tecnologias
- Backend: Node.js (TypeScript)
- Framework: Nest.js
- Frontend: HTML5, CSS3 e Javascript
- Banco de Dados: PostgreSQL
- Controle de Versão: Git + GitHub

## Funcionalidades principais
- Cadastro e autenticação de professores
- Gestão de instituições, cursos, disciplinas e turmas
- Importação de alunos via CSV
- Lançamento de notas
- Cálculo de notas finais (Media simples ou ponderada)
- Exportação de dados em CSV
